# Setup Guide

Follow these instructions to enable the full power of Brainback.

## 1. Stock Android (Pixel, Motorola)
1. Install the APK.
2. Go to **Settings > Accessibility > Downloaded Apps**.
3. Select **Brainback** and toggle ON.
4. Go to **Settings > Apps > Special App Access > Usage Access**.
5. Grant access to **Brainback**.

## 2. Samsung (One UI)
1. Install the APK.
2. Go to **Settings > Accessibility > Installed Apps**.
3. Enable **Brainback**.
4. To prevent battery optimization killing the service:
   - Go to **Settings > Apps > Brainback > Battery**.
   - Select **Unrestricted**.

## 3. Xiaomi (MIUI/HyperOS)
1. Install the APK.
2. Go to **Settings > Accessibility > Downloaded Services**.
3. Enable **Brainback**.
4. **Important**: Grant "Display over other apps" and "Auto-start" permissions in the app settings to ensure the lock timer persists.

## Troubleshooting
- **Timer Reset?** Ensure the app is not being "Cleared" from memory by system cleaners.
- **Blocking stopped?** Check the "Firewall" toggle on the main dashboard.
